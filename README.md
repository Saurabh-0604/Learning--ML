## 📚 Modules Covered

### 1. Python Fundamentals
Core Python programming concepts that form the foundation for all ML work — variables, data types, control flow, functions, OOP basics, file handling, and error handling.

### 2. Data Analytics & Visualization (Pandas, NumPy, Matplotlib, Seaborn)
Data manipulation, cleaning, and exploratory data analysis (EDA) using:
- **NumPy** — numerical computing and array operations
- **Pandas** — data wrangling, cleaning, and transformation
- **Matplotlib** — foundational plotting and charting
- **Seaborn** — statistical data visualization

### 3. Data Analytics Using AI Tools
Leveraging AI-powered tools to accelerate and enhance data analysis workflows — including automated EDA, insight generation, and AI-assisted data cleaning.

### 4. Supervised Learning – Regression
Predicting continuous values using algorithms such as Linear Regression, Polynomial Regression, Ridge/Lasso, and Decision Tree/Random Forest Regressors, along with model evaluation metrics (MAE, MSE, RMSE, R²).

### 5. Supervised Learning – Classification
Predicting categorical outcomes using algorithms such as Logistic Regression, KNN, Decision Trees, Random Forest, SVM, and Naive Bayes, with evaluation via accuracy, precision, recall, F1-score, and confusion matrices.

### 6. Unsupervised Learning – Clustering
Discovering patterns in unlabeled data using K-Means, Hierarchical Clustering, and DBSCAN, along with dimensionality reduction techniques like PCA.

### 7. AI Tools for ML Lifecycle Tracking
Tools and practices for tracking experiments, versioning models, and managing the ML lifecycle — covering concepts like experiment logging, model versioning, and reproducibility.

## 🚀 How to Use This Repository

1. Browse the folder corresponding to the topic you're interested in.
2. Each folder contains standalone scripts/notebooks with comments explaining the concept.
3. Follow the modules in sequence if you're learning ML from scratch — the structure mirrors a typical learning path from fundamentals → analytics → supervised learning → unsupervised learning → lifecycle tools.

## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/<your-username>/ML-Learnings.git
cd ML-Learnings
```

(Optional) Set up a virtual environment and install common dependencies:

```bash
python -m venv venv
source venv/bin/activate   # On Windows: venv\Scripts\activate
pip install numpy pandas matplotlib seaborn scikit-learn jupyter
```

Launch Jupyter Notebook to explore the code:

```bash
jupyter notebook
```
